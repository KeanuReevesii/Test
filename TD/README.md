# Unity初学时作品---塔防游戏demo

#### 介绍
完成度比较高的塔防小游戏，乃初学unity时期跟着教程做出来的产物

#### 使用说明

+ 使用wasd键可以移动视角，点击右侧炮台图标可以选择炮台种类，选好后点击地图白块可以建造
